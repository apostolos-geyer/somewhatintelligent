# somewhatintelligent — brand study

> Implementation status: accepted direction is captured in [`HANDOFF.md`](./HANDOFF.md).

This folder is a design study for turning `somewhatintelligent` into one coherent place for physical goods, software, writing, and Apostoli himself.

## Read in this order

1. [`01-brand-theory.md`](./01-brand-theory.md) — the core thesis, visual world, and rules.
2. [`02-interface-and-voice.md`](./02-interface-and-voice.md) — information architecture, UX, copy, and concrete page direction.
3. [`04-operational-system.md`](./04-operational-system.md) — real persistence boundaries, current capabilities, and the content-authoring extension.
4. [`HANDOFF.md`](./HANDOFF.md) — fresh-session scope, decisions, implementation sequence, and verification.
5. [`research/source-notes.md`](./research/source-notes.md) — what was inspected and what was taken from it.
6. [`reference-images/`](./reference-images/) — generated campaign/world-building images.
7. [`mockups/`](./mockups/) — generated interface concepts.

`source-assets/` contains the existing FRIEND shirt artwork used as visual reference. It contains no personal photographs.

## One-sentence brief

**somewhatintelligent is a publishing house for objects, systems, texts, and one unreliable human: institutional enough to feel consequential, personal enough to leak.**
